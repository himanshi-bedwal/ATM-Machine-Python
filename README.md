💳 ATM Machine Simulation

📌 Overview:

This is a Python-based ATM simulation project that performs basic banking operations. It is designed using a modular approach for better readability and structure.

🚀 Features:

- 💰 Deposit Money
- 💸 Withdraw Money
- 📊 Check Balance
- 📜 Transaction History

🛠️ Tech Used:

- Python
- Command Line Interface (CLI)

📂 Project Structure:

- main.py → Handles user interaction
- deposit.py → Deposit functionality
- withdrawl.py → Withdrawal functionality
- utils.py → Stores balance & transactions

▶️ How to Run:

1. Open the project
2. Run "main.py"
3. Follow the menu options

💡 Concepts Used:

- Functions
- Modules
- Conditional Statements
- List & Dictionary
